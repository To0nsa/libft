Detailed Description

Implementation of the get_next_line function.

This file provides an implementation of the get_next_line function, which reads a line (terminated by \n or EOF) from a file descriptor.

It supports multiple file descriptors in parallel by using a static array indexed by the file descriptor number. Each call returns one full line, including the newline character, if present.

Note
Requires a properly defined BUFFER_SIZE macro and MAX_FD for maximum number of file descriptors.
