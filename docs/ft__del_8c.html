Detailed Description

Simple delete function for use with linked list utilities.

This file contains the implementation of ft_del, a helper function designed to be passed as a deletion function to other linked list functions such as ft_lstdelone, ft_lstclear, or ft_lstmap. It simply frees the memory pointed to by the given pointer.
