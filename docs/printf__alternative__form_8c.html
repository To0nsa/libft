Detailed Description

Alternative form handling for hexadecimal output in ft_printf.

This file contains helper functions to determine when to apply the alternative form (#) and to prepend the appropriate prefix (0x or 0X) for hexadecimal output (x, X).

It also provides a function to calculate the length of the added prefix.
